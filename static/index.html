Alineación La alineación de las llantas es un proceso de mantenimiento necesario para asegurar que las llantas estén rectas y paralelas entre sí y con el suelo. Esto es importante para evitar un desgaste desigual de los neumáticos y garantizar una conducción segura del vehículo.
Balanceo El balanceo es el proceso de equilibrar el peso de las llantas y los rines. Aunque los neumáticos parecen iguales, pueden tener pequeñas diferencias de peso en cada lado. El balanceo se encarga de corregir estas imperfecciones mediante el uso de contrapesos colocados en el rin.
